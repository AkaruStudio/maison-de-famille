# Image

## Propriétés

---

| nom     | type                 | requis | default | description                |
| ------- | -------------------- | ------ | ------- | -------------------------- |
| sources | Tableau              | oui    | []      | Une liste de sources       |
| alt     | chaîne de caractères | oui    | ""      | Une description de l'image |

## API

---

```js
{
  sources: [__image__, __image_en_webp__],
  alt: 'Une maison'
}
```
